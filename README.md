# i茅台 Monitor Demo

这是一个合规的 i茅台 到货监测示例工程（演示模板）。

功能概述：
- 后端（FastAPI）：商品订阅、地址保存、模拟数据源、手动刷新与事件历史。
- 移动端（Expo React Native）：创建演示用户、保存地址、订阅商品、手动刷新与“打开官方客户端手动下单”的流程。

重要说明：
- 本仓库为合规演示模板。不包含任何未经授权的网页抓取、批量自动登录、绕过验证码或自动下单逻辑。
- 若你获得目标平台授权，可把官方 API 文档发给我，我将帮助你实现合法适配器。

目录结构（概要）：
- backend/: 后端代码（FastAPI）
- mobile/: 移动端示例（Expo React Native）

快速启动：请参见 backend/README.md 与 mobile/README.md
